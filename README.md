# tab UI

this library is browser tab module.

## install

### npm
```npm install --save-dev```

### bower
```bower install --save```


## usage

### browserify
```var tab = require('');```

### other
```
<script src="jquery.js"></script>
<script src="underscore.js"></script>
<script src=".js"></script>
<script src="app.js"></script> // tab load
```
